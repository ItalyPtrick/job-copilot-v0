## 学习内容（~1 小时）

深入理解 SQLAlchemy Column 类型映射：

| Python 类型 | SQLAlchemy Column | 说明 |
|---|---|---|
| `str` | `Column(String(50))` | 限长字符串 |
| `int` | `Column(Integer)` | 整数，`primary_key=True` 自动自增 |
| `dict/list` | `Column(JSON)` | 存储 JSON 结构化数据 |
| `datetime` | `Column(DateTime)` | 时间戳，可设 `default=datetime.now` |

关键参数：
- `nullable=False` — 必填字段，数据库层面约束
- `index=True` — 创建索引，加速按该字段查询
- `primary_key=True` — 主键，自动 autoincrement
- `default=` — Python 层面默认值（非数据库默认值）

`task_record.py` 参考 Doc 01 Step 3 的代码：

字段清单：
| 字段 | 类型 | 说明 |
|---|---|---|
| `id` | `Integer, primary_key` | 自增主键 |
| `task_type` | `String(50), index=True` | 任务类型（jd_analyze 等） |
| `payload` | `JSON` | 请求参数 |
| `status` | `String(20)` | success / error |

| `error_message` | `String(500)` | 错误信息 |
| `trace` | `JSON` | trace 事件列表 |
| `created_at` | `DateTime, default=datetime.now` | 创建时间 |
