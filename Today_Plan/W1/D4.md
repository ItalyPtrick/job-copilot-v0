## 学习内容（~1 小时）

### FastAPI 依赖注入

```python
# Depends(get_db) 让 FastAPI 自动管理数据库 session
@app.post("/task")
def create_task(db: Session = Depends(get_db)):
    # db 在请求结束后自动关闭
```

关键点：
- `get_db()` 是一个生成器函数（用 `yield`）
- FastAPI 在请求开始时调用 `next()`，结束时调用 `close()`
- 每个请求独立的 session，避免线程安全问题

### FastAPI Lifespan

```python
@asynccontextmanager
async def lifespan(app: FastAPI):
    # 启动时执行
    Base.metadata.create_all(bind=engine)
    yield
    # 关闭时执行（清理资源）
```

- 替代旧版 `@app.on_event("startup")`
- 开发环境用 `create_all`，生产环境用 Alembic
