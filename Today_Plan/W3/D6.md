## 学习内容（~30 分钟）

快速阅读 Doc 03 §4 Step 6 和 §5 的代码，理解：

- **双引擎解析**：规则引擎（正则提取时间/链接）+ AI 引擎（LLM 提取全部字段）
- **优先级策略**：规则结果优先（更可靠），AI 补充缺失字段（更灵活）

## 编码任务

### 任务 1：创建面试邀请解析器（Doc 03 Step 6）

创建 `app/modules/schedule/invite_parser.py`，参考 Doc 03 Step 6 的代码：

**函数 1：`parse_invite_rule_based(text)`**
- 用正则提取时间：`r"(\d{4}[-/]\d{1,2}[-/]\d{1,2}\s*\d{1,2}:\d{2})"`
- 用正则提取会议链接：腾讯会议 / Zoom / 飞书 3 种模式
- 返回 `{"start_time": "...", "meeting_link": "..."}` 或 None

**函数 2：`parse_invite_ai(text)`**
- 构造 prompt 要求 LLM 返回结构化 JSON：company / position / start_time / end_time / meeting_link / interviewer / notes
- 调用 `call_llm`

**函数 3：`parse_invite(text)`** — 双引擎入口
- 先调 `parse_invite_rule_based` 获取规则结果
- 再调 `parse_invite_ai` 获取 AI 结果
- 合并：规则结果优先，AI 补充缺失字段

**依赖**：
- `import re`
- `from app.services.llm_service import call_llm`

### 任务 2：创建 schedule 路由（可选，简化版）

创建 `app/modules/schedule/router.py`：

```python
from fastapi import APIRouter
from pydantic import BaseModel

router = APIRouter(prefix="/schedule", tags=["面试安排"])

class ParseInviteRequest(BaseModel):
    text: str

@router.post("/parse-invite")
async def parse_invite_endpoint(req: ParseInviteRequest):
    from app.modules.schedule.invite_parser import parse_invite
    return parse_invite(req.text)
```

在 `app/main.py` 中注册 `/schedule` 路由。

```python
def test_session_create_and_get():
    from app.modules.interview.session_manager import create_session, get_session
    session_id = create_session({"skill": "python_backend", "total_questions": 5})
    assert session_id is not None
    session = get_session(session_id)
    assert session["status"] == "created"
    assert session["config"]["skill"] == "python_backend"
```

**测试 2：`test_evaluation_extract_qa_pairs`**（纯逻辑，无需 API Key）
```python
def test_evaluation_extract_qa_pairs():
    from app.modules.interview.evaluation import _extract_qa_pairs
    messages = [
        {"role": "assistant", "content": "Python 的 GIL 是什么？"},
        {"role": "user", "content": "GIL 是全局解释器锁"},
    ]
    pairs = _extract_qa_pairs(messages)
    assert len(pairs) == 1
    assert "GIL" in pairs[0]["question"]
```

**测试 3：`test_invite_parser_rule`**（纯逻辑，无需 API Key）
```python
def test_invite_parser_rule():
    from app.modules.schedule.invite_parser import parse_invite_rule_based
    text = "面试时间：2025-04-20 14:00，腾讯会议链接：https://meeting.tencent.com/dm/abc123"
    result = parse_invite_rule_based(text)
    assert result["start_time"] == "2025-04-20 14:00"
    assert "tencent" in result["meeting_link"]
```
