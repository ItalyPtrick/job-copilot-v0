## 学习内容（~45 分钟）

新版 `/interview/answer` 不再“每次回答后直接出下一题”，而是模拟真人面试官的节奏：

```text
候选人回答
  ↓
planner 判断回答质量与追问价值
  ├─ should_follow_up = true  → 生成追问
  └─ should_follow_up = false → 进入下一道主问题
```

这比固定每题追问一次更自然：回答充分就推进，回答模糊或暴露误区就追问。

## 编码任务

### 任务 1：扩展 session 状态字段

在 session 中维护：

- `messages`：完整消息历史，每条 assistant 问题带 metadata。
- `questions_asked`：主问题列表，用于去重。
- `current_question_index`：已出主问题数。
- `current_main_question`：当前主问题结构。
- `current_follow_up_count`：当前主问题已追问次数。
- `covered_topics`：已覆盖考点。
- `recent_performance`：最近几轮回答质量摘要。

### 任务 2：创建 planner 函数

创建 `app/modules/interview/interview_planner.py`，实现：

- `plan_next_interview_action(session, latest_answer)`

返回结构：

```python
{
    "next_action": "follow_up" | "next_question" | "complete",
    "reason": "为什么这样决定",
    "follow_up_focus": "如果追问，追问什么",
    "next_difficulty": "easy|medium|hard",
    "performance_signal": "strong|normal|weak"
}
```

规则：

- 已达到 `total_questions` 且没有必要追问 → `complete`
- 当前主问题追问次数小于 `follow_up_count`，且回答有追问价值 → `follow_up`
- 否则 → `next_question`

难度调度规则：

- 根据 `total_questions` 和 `difficulty_distribution` 先生成基础主问题难度序列。
- 示例：`total_questions=5`、`{"easy": 0.4, "medium": 0.4, "hard": 0.2}` → `easy, easy, medium, medium, hard`。
- `performance_signal=strong` 时，下一题最多上调一档：`easy → medium`、`medium → hard`。
- `performance_signal=weak` 时，下一题最多下调一档：`hard → medium`、`medium → easy`。
- `performance_signal=normal` 时，沿用基础难度序列。
- 保护约束：不连续出 3 道 hard；不在 easy 配额未消耗时直接从 easy 跳到 hard。

### 任务 3：创建面试路由

`app/modules/interview/router.py`：

- `POST /interview/start`
  - 创建 session
  - 加载 skill 并构建 blueprint
  - 生成第一道主问题
  - 写入 session metadata
  - 返回 session_id + question

- `POST /interview/answer`
  - 读取 session
  - 记录候选人回答，按 D4 message contract 写入 metadata
  - 调用 `interview_planner.plan_next_interview_action` 决定下一步
  - 如果 follow_up：调用 `generate_follow_up`
  - 如果 next_question：调用 `generate_question`
  - 如果 complete：更新状态为 completed

- `POST /interview/evaluate`
  - 验证 session completed
  - 调用 `evaluate_interview(session["messages"])`
  - 写回 report

### 任务 4：注册路由

在 `app/main.py` 中注册：

```python
from app.modules.interview.router import router as interview_router
app.include_router(interview_router)
```
