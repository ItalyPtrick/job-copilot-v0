## 学习内容（~30 分钟）

### ReportLab 核心概念

```text
SimpleDocTemplate（页面配置：A4、边距）
    ↓ build([...])
Flowables（可排列元素）
    ├─ Paragraph（文本段落，支持样式）
    ├─ Table（表格）
    ├─ Spacer（间距）
    └─ Image（图片）
```

- `SimpleDocTemplate`：定义页面大小和边距
- `ParagraphStyle`：定义字体、字号、行距
- `build(elements)`：按顺序渲染所有 Flowable 到 PDF

### 中文字体问题

ReportLab 默认不含中文字体。解决方案：

```python
from reportlab.pdfbase import pdfmetrics
from reportlab.pdfbase.ttfonts import TTFont

# 注册字体（路径指向 .ttf 或 .ttc 文件）
pdfmetrics.registerFont(TTFont("Chinese", "C:/Windows/Fonts/msyh.ttc", subfontIndex=0))
```

本项目方案：
- **开发环境**：使用 Windows 自带微软雅黑 `msyh.ttc`
- **生产环境（Docker）**：下载开源思源黑体放入 `fonts/` 目录
- 代码中做 fallback：先找项目 `fonts/` 目录，找不到再用系统字体

**面试怎么讲**："ReportLab 默认不支持中文，需要手动注册 TTF 字体。开发环境用系统字体，生产环境打包开源字体，代码做优雅降级。"
