## 学习内容（~30 分钟）

### Celery 测试策略

测试 Celery 任务不需要启动 Worker，用 `task.apply()` 同步执行：

```python
def test_analyze_task():
    result = analyze_resume_task.apply(
        args=["test-id", "tests/fixtures/test_resume.txt", "Python开发"]
    )
    assert result.state == "SUCCESS"
    assert "basic_info" in result.result
```

`apply()` vs `delay()`：
- `delay()`：发送到 Broker，Worker 异步执行（需要 Worker 运行）
- `apply()`：在当前进程同步执行（不需要 Worker，适合测试）
