## 学习内容（~30 分钟）

### 结构化 Prompt 设计

让 LLM 返回固定结构 JSON，关键技巧：
1. System Prompt 中明确列出完整 JSON schema
2. 约束 "只返回 JSON，不要其他文字"
3. 容错：LLM 可能返回 markdown 代码块，需要 strip 后 parse

### 内容哈希去重

同一份简历重复上传时，无需再次调用 LLM：

```python
import hashlib

def content_hash(text: str) -> str:
    return hashlib.sha256(text.encode()).hexdigest()
```

数据库中 `(content_hash, target_role)` 加复合唯一约束，上传时先查是否已存在相同哈希 + 相同目标岗位。同一份简历换目标岗位需要重新分析（`match_analysis` 不同）。

**面试怎么讲**："用 SHA-256 对解析后的纯文本做哈希，配合 target_role 做复合唯一约束。相同简历 + 相同岗位秒返已有结果，换岗位则重新分析。"

## 编码任务

### 任务 1：扩展 ResumeRecord 模型

在 `app/database/models/resume.py` 已有骨架（W1-D7 创建，仅 `id` + `created_at` + TODO 注释）基础上扩展，不要新建文件。

修改 `app/database/models/resume.py`，补全字段：

```python
class ResumeRecord(Base):
    __tablename__ = "resume_records"

    id = Column(Integer, primary_key=True, autoincrement=True)
    resume_id = Column(String, unique=True, nullable=False, index=True)
    filename = Column(String, nullable=False)
    file_path = Column(String, nullable=False)
    file_ext = Column(String, nullable=False)
    content_hash = Column(String, index=True)
    # 复合唯一约束：同内容 + 同目标岗位才算重复
    __table_args__ = (UniqueConstraint("content_hash", "target_role", name="uq_resume_hash_role"),)
    status = Column(String, default="pending")  # pending/analyzing/completed/failed
    target_role = Column(String, default="")
    raw_text = Column(Text)
    analysis_result = Column(JSON)   # LLM 返回的结构化结果
    error_message = Column(String)
    created_at = Column(DateTime, default=...)
    updated_at = Column(DateTime, onupdate=...)
```

### 任务 3：创建 analyzer.py

创建 `app/modules/resume/analyzer.py`：

- `analyze_resume(resume_text: str, target_role: str = "") -> dict`
- System Prompt 要求返回固定 JSON：

```python
ANALYSIS_SCHEMA = {
    "basic_info": {
        "name": "姓名",
        "education": "学历",
        "years_of_experience": "经验年限",
        "skills": ["技能列表"],
    },
    "strengths": ["亮点1", "亮点2"],
    "weaknesses": ["不足1", "不足2"],
    "suggestions": ["建议1", "建议2"],
    "overall_score": 75,          # 1-100 整数
    "match_analysis": "与目标岗位匹配度分析（仅 target_role 非空时）",
}
```

- 复用 `app/services/llm_service.py` 的 `call_llm()` 函数
- 返回 dict（解析成功）或 `{"error": "..."}` （解析失败）
