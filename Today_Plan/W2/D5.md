## 编码任务

### 任务 1：集成到 Orchestrator（Doc 02 Step 6）

修改 `app/orchestrators/job_copilot_orchestrator.py`，添加 `_build_retriever_context` 函数：

```python
from app.modules.knowledge_base.vector_store import search
from app.types.retriever_context import RetrieverContext, RetrieverChunk
import uuid

def _build_retriever_context(query: str, collection: str = "default") -> RetrieverContext:
    docs = search(collection, query, top_k=3)
    chunks = [
        RetrieverChunk(
            chunk_id=f"chunk_{i}",
            source_title=doc.metadata.get("source_file", "unknown"),
            content=doc.page_content,
        )
        for i, doc in enumerate(docs)
    ]
    return RetrieverContext(
        context_id=str(uuid.uuid4()),
        status="success",
        chunks=chunks,
    )
```

**集成到 `execute_task`**：
- 在 Step 3（调用 LLM）之前，尝试构建 RAG 上下文
- 如果任务 payload 中有 `"use_rag": true` 和 `"rag_collection": "xxx"`，则调用 `_build_retriever_context`
- 将结果存入 `TaskResult.retriever_context`

**简化方案**（如果完整集成太复杂，可以先这样做）：
- 只实现 `_build_retriever_context` 函数
- 在 `execute_task` 的最终 `TaskResult.from_success()` 中，如果 payload 有 `use_rag`，则附上 retriever_context
- 不改变现有的 LLM 调用逻辑

### 任务 2：完善 knowledge.py 模型字段

当前 `app/database/models/knowledge.py` 只有 `id` 和 `created_at`。补充以下字段：

| 字段 | 类型 | 说明 |
|---|---|---|
| `filename` | `String(255)` | 原始文件名 |
| `collection_name` | `String(100)` | 所属向量集合名 |
| `file_path` | `String(500)` | 服务器上的存储路径 |
| `file_hash` | `String(64)` | 文件内容的 SHA256 哈希（防重复上传） |
| `chunks_count` | `Integer` | 分块数量 |
| `status` | `String(20)` | 状态：uploading / completed / failed |
| `file_size` | `Integer` | 文件大小（字节） |

示例：
```python
from sqlalchemy import Column, DateTime, Integer, String

class KnowledgeDocument(Base):
    __tablename__ = "knowledge_documents"

    id = Column(Integer, primary_key=True, autoincrement=True)
    filename = Column(String(255), nullable=False)
    collection_name = Column(String(100), nullable=False, default="default")
    file_path = Column(String(500), nullable=False)
    file_hash = Column(String(64), index=True)
    chunks_count = Column(Integer, default=0)
    status = Column(String(20), default="uploading")
    file_size = Column(Integer, default=0)
    created_at = Column(DateTime, default=lambda: datetime.now(timezone.utc))
```
