### 任务 1：编写单元测试（Doc 02 §5）

创建 `tests/test_rag_chain.py`，参考 Doc 02 §5 的代码：

**测试用例**：

| 测试函数 | 测试内容 | 断言 |
|---|---|---|
| `test_document_loader` | 加载 txt 文件并分块 | chunks 数量 > 1；每个 chunk 长度 ≤ 600 |
| `test_vector_store_add_and_search` | 向量存储的写入与检索 | 检索到 1 条结果；结果包含 "FastAPI" |
| `test_document_loader_unsupported_format` | 不支持的文件格式 | 抛出 ValueError |

**test_document_loader 实现**：
```python
def test_document_loader():
    from app.modules.knowledge_base.document_loader import load_and_split
    from pathlib import Path

    test_file = Path("tests/fixtures/test_doc.txt")
    test_file.parent.mkdir(exist_ok=True)
    test_file.write_text("这是测试文档。" * 200, encoding="utf-8")

    chunks = load_and_split(str(test_file))
    assert len(chunks) > 1
    assert all(len(c.page_content) <= 600 for c in chunks)
```

**test_vector_store_add_and_search 实现**：
```python
def test_vector_store_add_and_search():
    from app.modules.knowledge_base.vector_store import add_documents, search
    from langchain.schema import Document

    test_docs = [
        Document(page_content="Python 是一种编程语言", metadata={"source": "test"}),
        Document(page_content="FastAPI 是 Python 的 Web 框架", metadata={"source": "test"}),
    ]

    add_documents("test_collection", test_docs)
    results = search("test_collection", "什么是 FastAPI", top_k=1)
    assert len(results) == 1
    assert "FastAPI" in results[0].page_content
```

**test_document_loader_unsupported_format**：
```python
def test_document_loader_unsupported_format():
    from app.modules.knowledge_base.document_loader import load_and_split
    import pytest

    with pytest.raises(ValueError, match="不支持的文件格式"):
        load_and_split("test.xyz")
```

### 任务 2：完善集成测试

对齐当前 `tests/test_kb_api.py` 的真实覆盖范围，而不是从零新建一个最小草稿。

**当前应覆盖的重点**：

| 测试方向 | 当前关注点 |
|---|---|
| upload 成功 | HTTP 响应、`KnowledgeDocument` 记录、chunks_count 同步成立 |
| upload 失败 | 不支持格式返回 400；向量写入失败时清理文件与 DB；`db.commit()` 失败时补偿删除向量 |
| query | `/kb/query` 使用 JSON body，返回 `answer + sources` |
| collections | 直接反映 Chroma 当前 collection 状态 |
| stream | SSE 协议只承诺 `message/done` 两类事件 |

**实现要点**：
- 用测试专用 `FastAPI()` 挂载知识库 router，不直接复用 `app.main.app`
- 用 `dependency_overrides[get_db]` 把请求期 session 指到测试库
- `test_kb_api.py` 以 monkeypatch 为主，隔离真实向量库、模型与主应用 lifespan
- `/kb/query` 和 `/kb/query/stream` 都按 JSON body 发请求
